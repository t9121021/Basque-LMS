# Basque LMS
Simple Flask LMS prototype

## Install
Clone the repo:
git clone https://github.com/seabass42/Basque-LMS
cd Basque-LMS

Create virtual environment:
python3 -m venv venv
source venv/bin/activate

Install dependencies:
pip install -r requirements.txt

Run:
python run.py
